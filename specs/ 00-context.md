# Flowbit — Context

Flowbit is an Operations OS for SMEs. It tracks projects (sales/orders) through configurable stages (areas) with branching flows. Users have role-based permissions and stage-level access. The system is NOT formal accounting; it provides visual money tracking via simple income/cost events linked to projects.

Core goals:
- Visual control of project progress across stages
- Deadlines per stage and project timeline (planned vs actual)
- Unit quantity tracking per stage (qty_required, qty_done)
- Simple financial events: incomes and costs per project (margins)
- Products with BOM (materials per unit) to auto-calc project requirements (editable snapshot)
